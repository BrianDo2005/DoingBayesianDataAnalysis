Ex 4.1

\[ P(\theta = n) = 0.001 \]

\[ P(D = + | \theta = n) = 0.99 \]

\[ P(D = + | \theta = U) = 0.05 \]

Seeking \[ P(\theta = n | D = +)? \]

Bayes rules give

\[ P(\theta = n | D = +) = \frac{P(D = + | \theta = n) P(\theta = n)}{ P(D = +)} = \frac{P(D = + | \theta = n) P(\theta = n)}{ P(D = + | \theta = n)P(\theta = n) + P( D = + | \theta = U)P(\theta = U))} \]

So \(P(\theta = n | D = +) =\) 0.0194
